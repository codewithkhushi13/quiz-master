# QuizMaster – Multi-category Web Quiz

##  Description
QuizMaster is an interactive web-based quiz application that supports multiple categories such as General Knowledge and Technology. It provides a dynamic and engaging user experience with real-time feedback and score tracking.

##  Features
- User login system
- Timer-based quiz functionality
- Multiple categories (GK, Tech)
- Dynamic question loading
- Score tracking and high score storage
- Progress bar for quiz completion
- Option selection highlighting
- Restart quiz functionality
- Responsive design for mobile and desktop

##  Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
